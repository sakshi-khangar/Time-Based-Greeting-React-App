# Time-Based-Greeting-React-App
## Description
This React application displays a greeting message according to the current time of the user’s system. It shows “Good Morning”, “Good Afternoon”, or “Good Evening” using JavaScript time logic and React functional components.

## Features
- Time-based greeting message
- Clean and simple UI
- Uses React functional components
- Demonstrates conditional rendering
- Beginner-friendly React project

## Technologies Used
- React.js
- JavaScript
- Vite
- HTML5
- CSS3

## Learning Outcome
- Understanding React components
- Using JavaScript Date object
- Applying conditional logic in JSX
- Basic project structure in React

## Conclusion
This project successfully demonstrates how to use React functional components and JavaScript time logic to display dynamic content. By showing greetings based on the current system time, it helps in understanding conditional rendering, component structure, and real-time data usage in React. This project is simple, practical, and ideal for beginners to strengthen their React fundamentals.
